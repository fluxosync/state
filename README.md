# Fluxo

Fluxo é uma experiência curta.

Não é um jogo.  
Não é produtividade.  
Não é terapia.

É um estado temporário.  
Alguns sentem.  
Outros não.

Não há progresso.
Não há promessa.
Não há retenção forçada.

Use quando fizer sentido.
Encerre quando acabar.
